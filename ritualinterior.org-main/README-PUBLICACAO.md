## Fluxo do funil

1. Abra `index.html`.
2. Responda o quiz.
3. Confira a tela de análise energética.
4. Confira o resultado personalizado.
5. Clique em `Quero começar meu ritual agora`.
6. Confira o checkout interno da página.
7. O botão principal leva para `https://pay.kiwify.com.br/sIAKP0R`.
8. Para revisar a entrega, abra `paz-obrigado.html`.

## Observações de publicação

- Para GitHub Pages, publique o conteúdo desta pasta na raiz do repositório.
- O arquivo inicial do site deve permanecer como `index.html`.
- A página de obrigado deve permanecer como `paz-obrigado.html`.
- Configure a Kiwify para redirecionar o comprador para `https://ritualinterior.org/paz-obrigado.html` após o pagamento aprovado.
- CSS e JavaScript estão embutidos nos HTMLs para manter o carregamento simples e reduzir arquivos extras.
- Não há imagens locais nesta versão; a identidade visual atual é feita com tipografia, cores e elementos leves.
